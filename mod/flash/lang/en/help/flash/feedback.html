Flash Activity Feedback

This is the feedback which will appear on the results page and will be viewable by the user after a user has taken the activity.

Feedback : This is displayed when the user is logged in and has taken a test.

Feedback For a Guest User : If guest users are allowed to do this test then this feedback is displayed when any guest user has taken the test. It is recommended not to display feedback on attempts at a test that will possibly give away the answer if this is a guest user.

%Codes% for use in Feedback : Certain codes surrounded in percent signs will be replaced by values which have been communicated to Moodle from the Flash movie. Typically these will include :

  • %answer% - feedback about the answer that the learner gave.
  • and %time% - the time it took for the learner to answer.

For Flash MX learning interaction movies loaded and tracked with the preloader you can also use these codes :

  • %interaction_id% - the id that was given to the question when authoring the movie.
  • %interaction_type% - the type of the interaction.
