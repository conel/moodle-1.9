Flash Activity Module Admin Commands

Module admin commands for the Flash Activity : From the Flash module settings page the following commands are available :

  • Update flash_movies records : If necessary click on this link to resync the information in the database about movie size, framerate and background colours. Shouldn't be necessary unless you re-publish the movies with different sizes, framerates or background colours.
  • Clear event log : Delete log file.
  • Refresh event log display : Refresh the Flash event log display which is shown below these command links.
