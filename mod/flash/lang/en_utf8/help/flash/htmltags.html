HTML Tags Supported in Flash

Flash supports some simple html tags. And some of the HTML tags format text in a slightly different way to in normal html.

The tags in the following are supported by Flash :

<font size='32'><p align='center'>You 
can use some html tags 
here such as <b>bold</b> or <i>italics</i> 
or <u>underline</u>.</p></font>

Which would look something like this :

You can use some html tags here such as bold or italics or underline.

Don't forget inverted commas around attribute values.

You can often drop the " or ' inverted commas around values in normal html. For example this works in html :

<font size=32>Hello</font> 
<-- doesn't work in Flash -->

Flash is not so forgiving don't forget your inverted commas.

The <font> tag.

In Flash the font size is interpreted as a pixel size. It supports color attributes such as :

<font size="32" color="#00FF00">
This text is 32 pixels high and green</font>

Which would appear as :

This text is 32 pixels high and green.

The <img> tag.

The image tag is supported for loading non progressive jpgs and .swf files. But this only works for people who have the Flash 7.0 player installed.

The <p> tag.

You can use the paragraph tag to align text <p align="left"></p>,<p align="centre"></p> and <p align="right"></p> are supported.

<ol> (ordered list) and <ul> (unordered list) tags are ignored.

<a> Anchor Tag.

The anchor tag can be used to make a link. Normally it won't be coloured or underlined as it would be in normal html. But you can style anchors yourself as follows :

<a href="http://jamiep.org"><font color="#0000FF"><u>My site!</u></font></a>

Which would appear as :

My site!

Think carefully when using anchors within an activity as obviously a user clicking on an anchor will mean they abort the activity halfway through. You can use the target attribute though to open a link in a new window as follows :

<a href="http://jamiep.org" target="_blank" ><font color="#0000FF"><u>My site!</u></font></a>
