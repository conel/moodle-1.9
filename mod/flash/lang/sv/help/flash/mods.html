Hjälp för Flash-modulen

Flash-modulen gör det möjligt för författare av Flash-filmer att integrera sina produkter i Moodle som en aktivitet. Den gör det lätt att anpassa Flash-filmer, med ett minimum av kodning så att de:

  • kan användas som en aktivitet i en Moodle-kurs
  • kan logga aktiviteterna så att de dyker upp i Moodles block för Senaste aktiviteter och Moodles loggar
  • alternativt kan konstrueras så att man kan ändra syftet med dem (möjlig att konfigurera)
  • alternativt kan lagra betyg/omdömen i Moodles sida för statistik om detta
  • alternativt kan lagra svar och den tid det tar att besvara en fråga
  • visa användare detaljer om deras försök
  • alternativt kan låta (distans)lärare exportera betyg/omdömen från Moodles sida för statistik
  • konfigurationsdata, loggar, svar och betyg/omdömen kan säkerhetskopieras och återställas
  • en förladdare gör att filmer kan använda förladdade unicode fonter eller deluppsättningar av fonter som om de själva var inbäddade i filmerna

Mer information om hur Du skapar filmer att använda som Moodle-aktiviteter hittar Du på http://jamiep.org

Så som modulen t.v. är utformad så består aktiviteterna av ett flertal frågor som användare kan försöka besvara valfritt antal gånger.
