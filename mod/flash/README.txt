Flash Activity Module


Installation :
-------------
*Upload these files to yoursite/mod/flash/
* IF YOU INTEND TO REPUBLISH loader or order_and_select movies move files in flashclassfilespackage 
to a directory in a Flash class include path 
	eg. C:\Documents and Settings\Jamie\Local Settings\Application Data\Macromedia\Flash MX 2004\en\Configuration\Classes

A Moodle activity module that takes care of a lot of functionality for Flash
movie developers who want their movies to work with Moodle
to use Moodles grades table, configuration, backup and restore features etc.
