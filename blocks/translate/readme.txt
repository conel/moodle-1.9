Moodle Translate Block v1.0
Paul Holden, Greenhead College [pholden@greenhead.ac.uk]

Adds translation capabilities to a Moodle course, using upto 5 different languages.

It uses AJAX alongside the Google Translation API [http://google.com/translate_t].

Please read install.txt for installation instructions.

Tested in Moodle version 1.7.1

See http://gcmoodle.greenhead.ac.uk/external/translator/ for more information
